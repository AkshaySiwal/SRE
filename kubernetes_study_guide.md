
# Kubernetes Study Guide for Platform Engineer Interview

This study guide provides a curated list of resources to learn Kubernetes fundamentals for a Platform Engineer interview. The resources are categorized to help you focus on specific areas.

## 1. What is Kubernetes and Why It's Used

*   **Article:** [Kubernetes Tutorial for Beginners by Spacelift](https://spacelift.io/blog/kubernetes-tutorial) - A comprehensive introduction to Kubernetes, its features, and why it's the leading container orchestration platform.
*   **Video:** [Kubernetes in 5 minutes](https://www.youtube.com/watch?v=PH-2FfFD2PU) - A short, animated video that explains the basics of Kubernetes and its benefits.

## 2. Core Concepts

*   **Documentation:** [Kubernetes Basics](https://kubernetes.io/docs/tutorials/kubernetes-basics/) - The official documentation is the best place to learn about the core concepts. It provides interactive tutorials to learn about Pods, Deployments, Services, and more.
*   **Article:** [Kubernetes Core Concepts](https://praveendandu24.medium.com/kubernetes-tutorial-for-beginners-mastering-the-basics-in-1-hour-332db7b5916b) - A detailed blog post explaining the core Kubernetes objects with clear examples.

## 3. Kubernetes Architecture

*   **Article:** [Kubernetes Architecture Explained by DevOpsCube](https://devopscube.com/kubernetes-architecture-explained/) - A deep dive into the Kubernetes architecture, explaining the roles of the control plane, worker nodes, and all the components.
*   **Video:** [Kubernetes Architecture Explained by TechWorld with Nana](https://www.youtube.com/watch?v=umXE332_4pA) - A clear and concise video explaining the Kubernetes architecture with diagrams.

## 4. kubectl Commands and Practical Usage

*   **Cheat Sheet:** [Kubernetes Cheat Sheet by Spacelift](https://spacelift.io/blog/kubernetes-cheat-sheet) - A comprehensive cheat sheet with all the important kubectl commands.
*   **Tutorial:** [Kubernetes by Example](https://kubernetesbyexample.com/) - A collection of tutorials that teach you how to use kubectl to manage Kubernetes objects.

## 5. Container Orchestration Concepts

*   **Article:** [What is Container Orchestration?](https://www.redhat.com/en/topics/containers/what-is-container-orchestration) - An article by Red Hat that explains the concept of container orchestration and its importance.
*   **Video:** [Container Orchestration Explained](https://www.youtube.com/watch?v=kBF6Bvth0zw) - A video that explains container orchestration in a simple and easy-to-understand way.

## 6. Best Beginner-to-Intermediate Video Tutorials

*   **YouTube Channel:** [TechWorld with Nana](https://www.youtube.com/c/TechWorldwithNana) - This channel has a lot of high-quality videos on Kubernetes, from beginner to advanced topics.
*   **YouTube Channel:** [FreeCodeCamp.org](https://www.youtube.com/c/Freecodecamp) - This channel has a full-length course on Kubernetes for beginners.

## 7. Easy-to-Understand Blog Posts and Tutorials

*   **Blog:** [The Illustrated Children's Guide to Kubernetes](https://www.youtube.com/watch?v=42_vs_Ansyw) - A fun and easy-to-understand blog post that explains Kubernetes concepts using illustrations.
*   **Blog:** [Kubernetes for Beginners](https://www.redhat.com/en/topics/containers/learning-kubernetes-tutorial) - A series of blog posts by Red Hat that covers all the fundamentals of Kubernetes.

## 8. Hands-on Labs or Interactive Learning Platforms

*   **Platform:** [Play with Kubernetes](https://labs.play-with-k8s.com/) - A free in-browser Kubernetes playground that allows you to run a Kubernetes cluster in seconds.
*   **Platform:** [KodeKloud](https://kodekloud.com/free-labs/kubernetes) - KodeKloud offers free hands-on labs to practice your Kubernetes skills.

## 9. Common Kubernetes Interview Questions for Platform Engineers

*   **Article:** [Top 46 Kubernetes Interview Questions and Answers in 2024 by DataCamp](https://www.datacamp.com/blog/kubernetes-interview-questions) - A comprehensive list of Kubernetes interview questions with detailed answers.
*   **Article:** [40 Must-know Kubernetes Interview Questions and Answers by Simplilearn](https://www.simplilearn.com/tutorials/kubernetes-tutorial/kubernetes-interview-questions) - Another great resource for preparing for your Kubernetes interview.
